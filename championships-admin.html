🏆 Championship Logos

Drop PNG/SVG files into championships/logos/<sport-key>/ and list them in manifest.json.
Setup: for each sport, create a manifest.json file like: 
{ "files": ["primary.png", "knockout.png", "wordmark.png"] }
The public Championships page bundles those files into a single ZIP on-demand.

📤 Upload Destinations (Box File Requests)

Per-sport list of Box File Request URLs — shown as upload buttons on each sport tab.
Tip: In Box, open a destination folder → More options → Request files via email or shared link. Paste the resulting File Request URL below with a clear label (e.g. Broadcast Melts, Highlight Reel, Postgame Audio). Anyone with the link can upload to that folder without a Box account.
