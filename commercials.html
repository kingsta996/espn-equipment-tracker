Admin Panel

Manage commercial spots from the Dashboard tab. The Admin Panel is for sport-format uploads and reviewing activity.

PSA File Request URLs

Paste the Box File Request URL for each PSA category. Schools see the School PSA link from the login screen's Upload button; admins use the CUSA Produced PSA link from the Upload tab. Saved to Supabase — changes take effect for everyone immediately.

School Codes

Used to build the School PSA filename (<YEAR><CODE>_PSAH ≤ 15 characters). Keep it short — 3–5 uppercase alphanumeric. Auto-uppercased and stripped of non-alphanumerics. Saves on tab-out or Enter.

School PSA Advertisers

Type the Advertiser for each Active School PSA (typically the school name). Saved automatically when you tab out or hit Enter.

Sync Filenames from Box

Pulls each commercial's current name from Box (using the saved box_file_id) and updates the catalog if the file was renamed in Box. Skips rows without a Box file ID.

Upload Sport Formats (ESPN+)

Upload an updated ESPN+ commercial format Excel file (.xlsx). The file must have a "Format" sheet matching the ESPN+ layout. Stored locally in this browser.
